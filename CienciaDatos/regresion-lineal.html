Capítulo 4 Regresion lineal

El objetivo no es estimar y hacer inferencia sino hacer predicciones

Modelo lineal

\[\begin{equation} \tag{4.1} Y = X \beta + u \end{equation}\]

Método de Mínimos Cuadrados Ordinarios

Solución

\[\begin{equation} \tag{4.2} \hat{\beta} = min \sum_{i=1}^{n}e_i^2 \end{equation}\]

\[\begin{equation} \tag{4.3} \hat{\beta} = (X'X)^{-1}X'Y \end{equation}\]

Bajo los supuestos clásicos el estimador de MCO es el mejor dentro de los lineales e insesgados (MELI).

\[\begin{equation} \tag{4.4} R^2 = 1 - \frac{SRC}{SCT} \end{equation}\]

Nota: \(SCT\) es una magnitud fija, por lo tanto, \(MCO\) maximiza el \(R^2\).

¿Cómo hacemos para predecir \(Y\)?

\(Y_i = X_i^{'} \beta + u_i\) con \(i=1,...,n\)

\(\hat{Y_i} \equiv X_i^{'} \hat{\beta}\)

\(E(\hat{Y_i}) = X_i^{'} \beta\)

\(V(\hat{Y_i}) = X_i^{'} V(\hat{\beta})X_i = \sigma^2X_i^{'}(X'X)^{-1}X_i\)

Resultado: si el estimador \(\hat{\beta}\) es insegado y de varianza mínima, entonces \(\hat{Y_i}\) es un predictor insegado y de varianza mínima, ambos en la clase de estimadores/predictores lineales e insesgados.

Error cuadrático medio \((EMC)\):

Predecir requiere estimar \((\beta)\)

Descomposición sesgo varianza:

\[\begin{equation} \tag{4.5} EMC(\hat{\beta}) = E(\hat{\beta} - \beta)^2 \end{equation}\]

Intuición: cuan mal estima \(\hat{\beta}\) depende de cuan descentrado esta en relación a la verdad (sesgo) mas cuan disperso es relación a su propio centro (varianza).

\[\begin{equation} \tag{4.6} EMC(\hat{\beta}) = Sesgo^2(\hat{\beta}) + V(\hat{\beta}) \end{equation}\]

ECM y error de pronóstico

Relacion entre el error de estimación con el error de predicción.

EMC

\[\begin{equation} \tag{4.7} Err(\hat{Y}) = E(Y - \hat{f})^2 \end{equation}\]

Error de pronostico

\[\begin{equation} \tag{4.8} Err(Y - \hat{f}) = EMC(\hat{f}) + \sigma^2 \end{equation}\]

  1. Error reducible \((ECM)\) mas irreducible \((\sigma^2)\).
  2. Link predicción y estimación. Predecir correctamente \((Err(Y - \hat{f}))\) requiere estimar \((EMC(\hat{f}))\) correctamente.

Utilizando la descomposicion:

\[\begin{equation} \tag{4.9} Err(Y-\hat{f}) = \underbrace{Sesgo^2(\hat{f}) + V(\hat{f})}_{\text{ECM}} + \sigma^2 \end{equation}\]

  1. Machine learning: estrategias sesgadas pueden implicar una reducción drástica en la varianza.
  2. Puede ser que el mínimo ECM ocurra para predictores sesgados.
